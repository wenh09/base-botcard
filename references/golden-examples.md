# 黄金文案样例（标杆，写稿前对照仿写）

> 这些是从 58 张历史推广卡里挑出的**过关标杆**，已把真实文案内嵌进来——**skill 脱离本地语料也能用**。
> 用法：**先用下面「检索表」定位最像的 1-2 张**，再去对应样例**仿它的句式与节奏**，换成本次内容。不要凭空写。

## 🏷 标签体系（新增样例也按这套打标，保证可检索）

每张样例打这 6 维标签：
- **类型**：单点功能 / 多功能合集
- **子类**：常规 / AI / 系列 / 发布会
- **领域**：问卷表单 / 仪表盘数据 / 工作流消息 / 应用模式 / 其他
- **形态**：短亮点（适合左右并排）/ 长描述（必纵向）/ 多功能并列 / 分组编号
- **句式**：`:GoGoGo:`slogan / 告别X对仗 / 提问式痛点 / 量化 / 叙事slogan
- **配色**：blue / indigo

# 单点功能

## 样例 S1 · 问卷支持「电话号码」题型（教科书级简洁单点）

```
标题：问卷支持「电话号码」题型
标签：功能上新　配色：blue（非 AI）
slogan：:GoGoGo: 支持手机号格式自动校验、信息收集更真实有效

核心亮点
- 专属输入管控：仅支持电话号码字符输入，杜绝录入无效数据
- 自动格式校验：支持国内 / 国际手机号、固定电话格式自动校验，号码更真实有效
- 一键快速导入：通讯录一键导入填写，不用手动敲数字，填写超省心

尾注：本群已开启灰度，期待你的体验和反馈～
```
**为什么标杆**：标题 = 产品/功能 + 题型，零废话；slogan 一句话价值；3 个关键词等长无逗号、动宾（管控/校验/导入），描述都走"痛点→解法"。

## 样例 S2 · 仪表盘图表复制到飞书文档（"告别 X"对仗范例）

```
标题：仪表盘图表支持复制到飞书文档啦
标签：功能上新　配色：blue
slogan：🎉 告别截图，图表一键「粘贴」进文档，数据实时更新，让汇报更出彩

核心亮点
- 一键入档：告别手动截图，图表直接「粘贴」进文档
- 实时同步：告别过期数据，源表修改，文档秒级刷新
- 高清交互：告别模糊死图，原生无损，支持悬停看数
- 权限继承：告别越权访问，自动继承源仪表盘权限，流转更安全
```
**为什么标杆**：4 个关键词全 4 字、整齐；每条描述统一"告别 X，再给解法"对仗，读起来有节奏。注意：整卡只在 slogan + 描述里用"告别"，没滥用。

## 样例 S3 · AI 能力上新：单条记录智能总结（AI 单点句式）

```
标题：🎉 多维表格 AI 能力上新：单条记录智能总结
标签：内测中　配色：indigo + ai-common_colorful

核心亮点（核心价值）
- 内容提炼：快速总结关键信息，掌握数据核心
- 问题洞察：智能识别异常值与潜在问题
- 行动建议：基于上下文生成可落地的建议
```
**为什么标杆**：AI 类标题固定句式「多维表格 AI 能力上新：XX」；亮点 4 字关键词、动宾、短促有力。

# 多功能合集

## 样例 M1 · AI 三件套（多功能并列标杆，本 skill 渲染样本同款）

```
标题：多维表格 2026｜AI 生成问卷、AI 问数、AI 生成组件邀你体验 🔥🔥
标签：功能上新　配色：indigo + ai-common_colorful
slogan：:GoGoGo: 无论是专业数据分析、高级图表生成，还是复杂问卷搭建，AI 都能帮你完成

功能块（每块 = 功能名 + 一句价值）
- AI 生成问卷：不止于生成，AI 将深入洞察业务场景，自主规划问题、显示逻辑和设置，让专业问卷在对话间即刻交付
- AI 问数：查数据、看指标、做分析。只需一句提问，即可获得专业的数据统计与分析结果，并清晰呈现分析过程
- AI 生成仪表盘组件：通过对话让 AI 生成专业图表组件，桑基图、矩形树图、热力图、瀑布图等任你探索，支持下钻查看明细，体验比肩原生组件

按钮（等分三列）：了解 AI 生成问卷 / 了解 AI 问数 / 了解 AI 生成组件
```
**为什么标杆**：合集标题 = 产品名 + 年份｜并列功能名 + 🔥🔥；slogan 一句总括全部功能；每个功能块"名 + 一句"；功能数 = 按钮数，等分。对应 `../samples/B_AI三件套_1大2小.card`。

标题：多维表格 2025 发布会首批功能，火热来袭！ 🔥🔥

  如今，发布会首批功能火热来袭！立即加入内测，成为第一批尝鲜新能力的效率先锋

# 用这些样例的纪律
- **先选最像的一张，仿句式**，再替换内容；不要把多个样例的风格混搭。
- 仿的是**结构与节奏**（标题句式、slogan 长度、关键词字数、描述对仗），不是照抄词。
- 仿完仍要过 [copy-checklist.md](copy-checklist.md) 的黑名单自检。
- 新攒的标杆（你或未来使用者贡献的过关卡）继续往这里加：① 按 archetype 归到对应段 ② **在「检索表」加一行 + 按「标签体系」6 维打标**（否则将来捞不到）③ 若示范了新句式/新结构，顺手更新 [copy-rules.md](copy-rules.md)。
