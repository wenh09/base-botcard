# 文案怎么写（导向 · 运行时主用）

> 配套：硬自检看 [copy-checklist.md](copy-checklist.md)；仿写标杆看 [golden-examples.md](golden-examples.md)；原始 58 张分析归档在 [research-notes.md](research-notes.md)。

## 0. 先判 archetype（决定句式）

只有 2 类（判定：**推几个功能**）：

| 类型 | 标题句式 | slogan | 内容结构 |
|---|---|---|---|
| **单点功能**（1 个，含 AI/系列） | `产品名 + 功能 + 价值动词`（AI 卡常用「多维表格 AI 能力上新：XX」；系列「💥…第 X 弹：YY」） | 1 句价值，AI 卡前缀 `:GoGoGo:` | 核心亮点(2-4) + 适用场景(可选) |
| **多功能合集**（2+ 个，含发布会） | `产品名 年份｜并列功能名，召唤 🔥🔥` | 1 句总括全部 / 发布会型 2-3 行叙事 | N 个功能块（名+一句）/「N」编号分组 |

AI / 系列 / 发布会都是"皮肤"，不是独立结构。仿写时翻 golden-examples 对应样例。

## 1. 亮点条目格式（近 100% 命中）

```
[emoji?] **关键词(4-7字)**：描述(15-30字)
```
- 关键词加粗，描述走"旧痛点 → 新解法"对比
- 同组关键词字数尽量一致、不带逗号（详见 checklist #1）
- 中文冒号 `：`

正例：`专属输入管控：仅支持电话号码字符输入，杜绝录入无效数据`

## 2. 痛点引入（三手法）

- **提问式**（强 hook，合集模板必用）：`订单总延期？` 紧接解法
- **「告别 X」**：`告别截图` / `告别从零搭建`（每卡 ≤ 1 处）
- **比喻化承诺**（品牌味）：`任督二脉` / `搬回家` / `专属数字工厂`（每卡 ≤ 1 个，标题或 slogan 点睛）

## 3. 量化承诺（提可信度）

有数字用数字：`30 多款`、`200 万行`、`100 个上限`、`360° 多维`。别笼统"很多/提升"。

## 4. 标签 tag + 尾注（PM 拍板逻辑）

**两件事分开、要问用户、别猜。**

标题 tag（`text_tag_list`，颜色只用 **green / blue**）：
| 放量节奏 | tag |
|---|---|
| 很快全量 | `功能上新` |
| 小范围逐步放量 | `内测中` |

底部尾注（footer div）：
| 卡片目的 | 尾注 |
|---|---|
| 邀请进群体验 | 进群开通内测，期待你的反馈和建议 |
| 已灰度到本群 | 本群已开启内测，期待你的反馈和建议 |
| 已完全上线（通知） | 功能已正式上线，欢迎体验和反馈 |

## 5. CTA 链接命名（按场景）

| 场景 | 文案 |
|---|---|
| 文档 | 帮助文档 / 查看帮助文档 / 快速上手秘籍 |
| 反馈/进群 | 加入内测群 / 加入反馈群 / 我要反馈 |
| 模板获取 | 👉 获取同款 / 🔥 一键获取模板 |
| 学习入门 | 立即开启学习之旅 |
| 系列反链 | 点我查看历史更新 |

## 6. emoji 规则

- unicode：🎉(上线庆祝) / 🔥🔥(重磅邀请，必双出) / 💥(系列第X弹) / ✨ / 🚀 / 💡。整卡 ≤ 8 个。
- 飞书 shortcode（仅飞书内渲染）：`:GoGoGo:`(AI slogan 前缀最高频) / `:WOW:` / `:Loudspeaker:` 等。导出非飞书环境会显示 `:xxx:` 原文。

## 7. 称呼语

默认「你」(~90%)；正式/招募用「您」；「我们」自指团队。一张卡不混用。

## 8. 固定句式（可直接用）

| 场景 | 句式 |
|---|---|
| 提供配套资料 | 我们还为你准备了 XX，快来试试吧 |
| 招募开头 | 亲爱的 XX 用户，诚挚邀请你加入 XX |
| 系列反链 | 想了解历史 XX 升级？点我查看历史更新 |
| 模板合集 slogan | 告别从零搭建｜精选 XX 高频实战模版，一键复用行业标准 |
